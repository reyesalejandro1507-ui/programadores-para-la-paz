Sistema operativo:
Es el software que administra los recursos del computador y permite ejecutar programas. 
También controla el hardware y facilita la interacción entre el usuario y las aplicaciones.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. 
Se utiliza principalmente para crear servidores y aplicaciones del lado del servidor.

Aplicación server.js:
Es el archivo que contiene el código del servidor desarrollado con Node.js y Express. 
Define las rutas y la forma en que el servidor responde a las solicitudes de los usuarios.

Puerto:
Es un punto de comunicación que permite el intercambio de datos entre el servidor y los clientes. 
En este caso, el servidor utiliza el puerto **3000** para recibir las solicitudes.

Logs:
Son registros de eventos y mensajes generados por el sistema o la aplicación. 
Sirven para verificar el funcionamiento del servidor y facilitar la detección y solución de errores.
